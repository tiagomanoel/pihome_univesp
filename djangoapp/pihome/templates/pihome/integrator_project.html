Projeto Integrador Univesp

O Projeto Integrador da Universidade Virtual do Estado de São Paulo (Univesp) é uma iniciativa acadêmica que visa integrar os conhecimentos adquiridos pelos alunos ao longo do curso em um projeto prático e interdisciplinar. Ele é uma oportunidade para os estudantes aplicarem conceitos teóricos em situações reais, promovendo o desenvolvimento de habilidades técnicas, criativas e colaborativas.

Objetivos do Projeto Integrador

  • Promover a integração entre as disciplinas do curso, permitindo uma visão mais ampla e prática do aprendizado.
  • Desenvolver soluções inovadoras para problemas reais, utilizando tecnologias modernas e acessíveis.
  • Estimular o trabalho em equipe, a comunicação e a colaboração entre os alunos.
  • Preparar os estudantes para os desafios do mercado de trabalho, com foco em projetos práticos e aplicados.

Sobre o Projeto PiHome

O PiHome é um exemplo de projeto desenvolvido no contexto do Projeto Integrador. Ele consiste em uma solução de automação residencial que utiliza tecnologias como MQTT, Django, Docker e PostgreSQL para oferecer uma plataforma eficiente e segura para o controle de dispositivos IoT.

Este projeto foi desenvolvido com o objetivo de proporcionar uma experiência prática aos alunos, permitindo que eles explorem conceitos de desenvolvimento de software, redes, segurança e integração de sistemas.

Impacto e Benefícios

  • Proporciona aos alunos uma experiência prática e interdisciplinar.
  • Contribui para o desenvolvimento de soluções tecnológicas inovadoras.
  • Promove a colaboração e o aprendizado em equipe.
  • Prepara os estudantes para os desafios do mercado de trabalho.

O Projeto Integrador é uma parte essencial da formação acadêmica na Univesp, permitindo que os alunos consolidem seus conhecimentos e desenvolvam habilidades práticas que serão fundamentais para suas carreiras.
